# 勤怠管理サイト「サボらん」
勤怠管理をしようよ。  
サーバーサイドは[こっち](https://github.com/kiesproject/kies-attendance)

## DevelopEnvironment
- React.js
- TypeScript
- Redux

## Running Project
1. Install node modules
```
$ npm install
```

2. Running dev server 
```
$ npm start

> parcel ./src/index.html

```
